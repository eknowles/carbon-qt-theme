# Carbon Qt Theme

> Builds a stylesheet and resource files for Qt 5

## Setup
### Demo Application

```bash
sh setup
```

### Node.js

1. Install Node.js
1. Install Yarn

## Development

Edit `./theme/src/root.scss`

### Run Demo Application

It uses PyQt, first compile the stylesheets with `node build.js` in `./theme` dir. Then run `sh run` in root dir.
